โปรเจคเด็กฝึกงาน ระบบยืมคืนอุปกรณ์ (Equipment Borrowing System)

## 📖 เกี่ยวกับโปรเจกต์
เว็บแอปพลิเคชัน Full-stack สำหรับจัดการระบบยืม-คืนอุปกรณ์ ครอบคลุมการทำงานตั้งแต่การเข้าสู่ระบบ การจัดการข้อมูลของแอดมิน และการดูประวัติการทำรายการ

## ✨ ฟีเจอร์หลัก
- **ระบบเข้าสู่ระบบ:** ลอจิกการล็อกอินที่ปลอดภัยและตรวจสอบการยืนยันรหัสผ่าน
- **แดชบอร์ดแอดมิน:** หน้าจัดการข้อมูลอุปกรณ์ สถานะ และสิทธิ์ของผู้ใช้งาน
- **ประวัติการยืมคืน:** ตรวจสอบข้อมูลการยืมคืนย้อนหลังและสถานะปัจจุบัน
